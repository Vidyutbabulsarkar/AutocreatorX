AutoCreatorX - Deployment Guide
-----------------------------

1. Prerequisites:
   - Python 3.9+
   - pip install streamlit

2. How to Run Locally:
   a. Save files in a folder (AutoCreatorX)
   b. Open terminal in the AutoCreatorX folder
   c. Run: streamlit run app.py

3. Deploy for Free:
   a. Go to https://share.streamlit.io
   b. Connect your GitHub repo or upload a ZIP file of this project
   c. Launch your app online
